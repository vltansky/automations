# Codex Automations

Shared Codex automation packages.

## Install

```bash
npx -y codex-automations add vltansky/automations --list
npx -y codex-automations add vltansky/automations --automation <id> --dry-run
npx -y codex-automations add vltansky/automations --automation <id>
```
